# Remote Monitoring Tool - Educational Project

## 📋 Overview
This tool demonstrates client-server remote monitoring with Windows authentication. It's designed for educational purposes to understand how remote access systems work.

## ⚙️ Features
- ✅ Windows system authentication (use your Windows login)
- ✅ Secure password hashing
- ✅ Client-server socket communication
- ✅ Remote command execution (whitelisted commands)
- ✅ System information gathering
- ✅ Connection logging
- ✅ Background website loading (when compiled as EXE)
